Here you will find the required files to built a low-cost EPTS.
https://ieeexplore.ieee.org/abstract/document/9476415
Folder device:
- It contains the inventory to built your device. Also contains required files to print a 3D case.
- It also contains an example of file to download stored data from the SD of each device to computer. Be free to modify it.

The following folders are intended to be used together. For further information see https://www.mdpi.com/2079-9292/13/10/1876

SampleData:
- Example of data formatted to analyze

PythonScripts:
- It contains examples of how to analyze and extract information from SampleData

SampleImage:
- It contains examples of images obtained after applying PythonScripts.
